# Punky Roadmap

## v0.1.0-alpha (current)
- [x] Rich text editor (bold, italic, underline, strikethrough)
- [x] Headings H1–H3
- [x] Bullet and numbered lists
- [x] Pull quotes / blockquotes
- [x] Code blocks and inline code
- [x] Link insertion
- [x] Image insertion (URL, file picker, drag & drop)
- [x] YouTube and Vimeo embed (auto-parse URL)
- [x] Custom HTML / script injection
- [x] GitHub API publish (create + update)
- [x] index.json manifest auto-maintenance
- [x] Created / modified timestamps
- [x] Local draft auto-save (localStorage)
- [x] Export standalone HTML file
- [x] Preview mode
- [x] Post metadata: slug, excerpt, category, tags
- [x] Blog index page (reads manifest, category filters, pagination)
- [x] Settings persisted across sessions

---

## v0.2.0 — Quality of life
- [ ] Keyboard shortcuts (Ctrl+B, Ctrl+I, Ctrl+K for link, etc.)
- [ ] Image resize handles in editor
- [ ] Post list / history panel (load previous posts from manifest for editing)
- [ ] Word count and read-time estimate
- [ ] Slug auto-sanitize (strip special characters on input)
- [ ] Better mobile editing experience
- [ ] Dark/light preview toggle

---

## v0.3.0 — Blog features
- [ ] RSS feed generation (auto-maintained alongside index.json)
- [ ] Open Graph image field per post
- [ ] Featured post pinning
- [ ] Search on blog index page (client-side, no server)
- [ ] Tag filter on blog index page
- [ ] Reading progress indicator on post pages

---

## v0.4.0 — Power features
- [ ] Multi-repo support (publish to different blogs from one editor)
- [ ] Schedule publish (set future date, reminder to manually push)
- [ ] Post templates
- [ ] Import from Markdown file
- [ ] Export to Markdown
- [ ] Trojan Horse native wrapper support (filesystem access, no token needed for local repos)

---

## Someday / Maybe
- Table insertion
- Footnotes
- Math/LaTeX rendering (KaTeX)
- Syntax highlighting in code blocks (Prism.js)
- Collaborative editing (stretch goal — requires backend)
